> checks a variable for a condition, gives value back or a defined value

## Install

```sh
npm i -D ss-checkvar
```

## Usage

```js
import ssCheckvar from "ss-checkvar"

ssCheckvar(myVarToCheck, "return this string when empty") // true
```
